### 安装Redit
##### 安装gcc编译器
```
yum install gcc-c++
```	
##### 解压源码包
```
tar -zxvf redis-3.0.7.tar.gz
```
##### 进入解压后文件夹的根目录执行编译
```
make
```
##### 安装Redis
```
make PREFIX=/usr/local/redis install
```
##### 将解压文件夹中的redis.conf文件复制到安装目录
```
cp redis.conf /usr/local/redis
```

##### 启动redis
```
./bin/redis-server ./redis.conf
```
##### 检测是否启动了redis
```
ps -ef | grep -i redis
```
##### 停止redis
```
./bin/redis-cli shutdown
```

##### redis命令行操作
* set key value 
* get key
* keys *
* del key
